# React payment terminal prototype

## Get Started

1. **Clone the project**

    `git clone https://github.com/Nalyvaiko/payment-terminal.git`.

2. **Install dependencies**

    `npm install`

3. **Run the example app**

    `npm start -s`

## Technologies used

    **React**
    **Redux**
    **React Router**
    **Webpack**
    **Jest**
    **Enzyme**
